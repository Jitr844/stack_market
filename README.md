# 📈 Stock Market Dashboard

A simple and responsive **Stock Market Dashboard** built using **HTML, Bootstrap, and JavaScript**. The application displays real-time stock market data, including stock prices, daily changes, and market trends through an easy-to-use interface.

---

## 🚀 Features

- 📊 View real-time stock prices
- 🔍 Search stocks by company name or symbol
- 📈 Display daily price changes
- 💹 Market trend visualization
- 📱 Responsive design with Bootstrap
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- Bootstrap 5
- JavaScript (ES6)
- Stock Market API (e.g., Alpha Vantage, Finnhub, Twelve Data)

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/jitr844/stock-market.git
```

### 2. Navigate to the project folder

```bash
cd stock-market
```

### 3. Add your API Key

Open `app.js` and replace:

```javascript
const API_KEY = "YOUR_API_KEY";
```

### 4. Run the project

Open `index.html` in your browser.

---

## 📖 How to Use

1. Enter a stock symbol (e.g., **AAPL**, **GOOGL**, **TSLA**).
2. Click the **Search** button.
3. View the latest stock price and market information.
